# Espósito Pizzeria — Brand

Brandbook e biblioteca de assets da Espósito Pizzeria, Barão Geraldo · Campinas.

## Como usar

Abra o `index.html` em qualquer navegador (Chrome, Safari, Firefox).
Para imprimir/exportar PDF: `Ctrl+P` (Windows) ou `Cmd+P` (Mac).

## Estrutura

```
.
├── index.html          ← brandbook completo (10 seções)
└── assets/             ← logos, ícone, ilustração, fotos
```

## Assets disponíveis

**Logo (SVG vetorial)**
- `logo.svg` — versão colorida principal
- `logo-mono-white.svg` — branco para fundos escuros
- `logo-mono-red.svg` — monocromático vermelho
- `logo-mono-dark.svg` — monocromático escuro

**Ícone (coroa + e)**
- `icone.png` — colorido
- `icone-branco.png` — branco

**Ilustração**
- `ilustracao.jpeg` — "...è fatta la pizza" (Raffaele Espósito, 1889)

**Fotografia**
- `pizza-01.png` a `pizza-05.png` — pizzas
- `ambiente-01.png` a `ambiente-07.png` — ambiente, forno, mesa
